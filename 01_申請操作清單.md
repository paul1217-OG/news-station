# 申請操作清單｜每日新聞站上線前的 5 個必開分頁

> 依序點開、依序完成。每完成一個就回頭跟我說「第 X 步完成」，我會給你下一步的指示。

---

## ✅ 步驟 1：申請 LINE 官方帳號（10 分鐘）

🔗 **點我開啟：** https://www.linebiz.com/tw/entry/

操作：
1. 點「免費試用」
2. 用個人 LINE 帳號登入（或新建商用帳號）
3. 帳號類型選 **「未認證帳號」**（免費，事後可升級認證）
4. 帳號名稱：自由命名（建議：你的新聞站名稱，例如「Paul 的財經早報」）
5. 行業選「**媒體與資訊**」
6. 完成 → 自動進入 LINE Official Account Manager

**完成後，把官方帳號 ID（@xxx 開頭）貼給我。**

---

## ✅ 步驟 2：啟用 Messaging API（5 分鐘）

🔗 **點我開啟：** https://manager.line.biz/

操作：
1. 進入剛建立的官方帳號
2. 右上「**設定**」→ 左側選單「**Messaging API**」
3. 點「**啟用 Messaging API**」
4. 系統要求建立 **Provider（提供者）**：自由命名（例：`paul-news-provider`）
5. 同意條款 → 完成

---

## ✅ 步驟 3：取得 Channel Access Token（5 分鐘）

🔗 **點我開啟：** https://developers.line.biz/console/

操作：
1. 登入後，找到剛建立的 Provider → 點進去
2. 進入 Messaging API Channel
3. 切換到「**Messaging API**」分頁
4. 找到「**Channel access token (long-lived)**」→ 點 **Issue**
5. 把出現的長字串 **複製存好**（這是密碼等級的東西，不要外流）
6. 順便往下滾，找到「**Channel secret**」也複製存好

⚠️ **回頭把這兩個值貼給我（在私訊環境，注意安全）：**
- `Channel access token`：?
- `Channel secret`：?

---

## ✅ 步驟 4：申請 OpenAI API Key（5 分鐘）

🔗 **點我開啟：** https://platform.openai.com/api-keys

操作：
1. 用 Google / Microsoft 帳號登入（或註冊新帳號）
2. 左側「**Billing**」→ **新增信用卡 + 加值至少 USD 5**（最低門檻）
3. 回到「**API keys**」→ 點「**Create new secret key**」
4. 命名：`news-station-prod`
5. 把 `sk-xxxxxx...` **複製存好**（只會顯示一次）

> 💰 **預算說明：** GPT-4o-mini 每日 1 次摘要約消耗 $0.01 USD（NT$0.3）。USD 5 可用約 1.5 年。

---

## ✅ 步驟 5：註冊 n8n Cloud（5 分鐘）

🔗 **點我開啟：** https://app.n8n.cloud/register

操作：
1. 用 email 註冊（建議用 Gmail，方便日後接 Google 服務）
2. 14 天免費試用，**先不用刷卡**
3. 工作區命名：`news-station`
4. 試用期間決定要不要付費（NT$600/月起）

> 💡 **替代方案：** 如果想完全免費起步，可以選 Make.com（https://www.make.com/en/register），1,000 次/月免費操作。但 n8n 介面對台灣使用者較友善。

---

## ✅ 步驟 6（選用）：公開資訊觀測站 OpenAPI 探勘（不用註冊）

🔗 **點我開啟：** https://openapi.twse.com.tw/

操作：
1. **不需要註冊**（這是政府公開資料，免 API key）
2. 瀏覽：「上市公司每日重大訊息」相關 endpoint
3. 直接複製 endpoint URL，等下我會幫你寫進抓取程式
